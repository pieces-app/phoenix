# 1.5.2: Better code language detection, a new Information View and more!
> **Published by the Pieces Team** | May 1st, 2023

Pieces for Developers gets a heavy upgrade today as we start to polish the UI and roll out updated machine learning models. We’d love your feedback - just hit [ Shift + ! ] from the Pieces app or email support@pieces.app.

## Pieces for Developers
**Pieces for Developers is the core user-facing Pieces application. Get started for free <a href="http://code.pieces.app/install" target="_blank" rel="noopener noreferrer">here</a>.**

### Adding Pieces
- Language classification - Pieces auto-classifies your snippets by language using machine learning models. In this release, we pushed a couple major upgrades: Improved code language classification predictions We now display the confidence level on our code language classification
- Better image error handling - you’ll now receive an error message when trying to upload an unsupported image type
- Image size constraints - images uploaded to the app must be under 1MB

![](https://uploads-ssl.webflow.com/6143afec68f555387049efb3/630d685a60912c0d284e5884_ezgif_com-gif-maker7_13b38a536756e887e64630fb3313fca5_800.png)

### Managing Pieces
- Information View - view more information related to your snippets in one handy view, including its description, related links, tags and origin
- Fixed an off-by-one bug with code selections when copying

![](https://uploads-ssl.webflow.com/6143afec68f555387049efb3/630d685960912c64514e5812_txn7lylaunebxq2440bp5jt7huq2_fd0647b9be6830ca5508fa51d4fe53b3.gif)

### Installing & Configuring Pieces
- PFD will stay running even when the user hits the close button on the window
- Up to 35% faster bootup performance to get you into the app faster
