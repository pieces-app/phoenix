# Pieces Release 1.3.9 - Integrations, better search and much more!
> Mon Mar 27 2023 13:40:51 GMT+0000 (Coordinated Universal Time)

We&#x27;re shipping a doozy of a release today! Here&#x27;s what to look forward to:

## Pieces for Developers Updates
<em>For newer users, Pieces for Developers is the core user-facing Pieces application.</em>

### Managing Your Account
- View and manage your login status from the Settings menu
- Tighter coupling between Pieces and Pieces OS
- Improved user messaging around failed connections between Pieces and Pieces OS

### Signup, Install and Onboarding
- Improved installation &amp; updating infrastructure
- New user onboarding experience upon installation and first initialization

### Viewing and Managing Your Pieces
- Users can now edit the description for a Piece
- Users can now reclassify the code language for a Piece   
- Users can now reclassify text as code or code as text   
- Improved keyboard shortcuts   
- Improved tooltips   
- Users can now scroll action menus using keyboard arrows   
- Code snippet previews now have line numbers   
- Scrollbars are now always present in action menus to denote scrollable  content  
- Removed wrapping from code previews in a narrow pane experience

### Search
- Upgraded search infrastructure to provide more accurate results 
- Search string highlights when Pieces returns an exact match on a search

## Pieces Integrations
- Launched code.pieces.app/integrations for users to add Pieces integrations

### JetBrains
- Users can now save pieces from within any JetBrains product using the right-click context menu or a hotkey (default is CMD+SHIFT+P) 
- Users can configure hotkeys

### Chrome
- Users can now save pieces from any web URL within Chrome using the right-click context menu or a hotkey (default is CMD+SHIFT+P)<br/>

## Pieces OS Updates
<em>For newer users, Pieces OS is the background service that enables Pieces to work locally and handles communications with any integrations you install, among many other tasks.</em>

### New Pieces OS UI, accessible via the MacOS toolbar
- Sign in/out of Pieces
- Learn more about Pieces Integrations   
- Manually quit Pieces OS   
- Manually check for updates<br/>

### More Updates
- Improved app installation and updating infrastructure
- Upgraded API endpoints to accommodate more <a href="https://code.pieces.app/integrations">Pieces Integrations</a>
