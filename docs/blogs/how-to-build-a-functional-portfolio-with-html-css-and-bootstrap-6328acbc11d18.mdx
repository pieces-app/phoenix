# How to Build a Functional Portfolio with HTML, CSS and Bootstrap
> Wed Apr 05 2023 16:53:15 GMT+0000 (Coordinated Universal Time)
<p>Today, whenever your average person thinks about the creation of a website or application, more complex languages like<a href="https://www.w3schools.com/js/default.asp" target="_blank" rel="noopener noreferrer"> <u>Javascript</u></a> and<a href="https://www.w3schools.com/python/default.asp" target="_blank" rel="noopener noreferrer"> <u>Python</u></a> are top of mind. However, these languages and their frameworks are a bit too complicated for beginners, especially when they‚Äôd only like to build simple websites, like portfolios, or static responsive websites, like landing pages.<p.new.line><p>HTML and CSS to the rescue! For the newbie or average developer who wants to create a portfolio website but isn&#x27;t very excited about the complex ideas that come with other programming frameworks, HTML and CSS are great. They allow you to create your websites with HTML and improve or style them with CSS. Of course, you can build things with them other than a portfolio, including:<p.new.line><ul><li>Interactive resumes<li.new.line><li>Email newsletters<li.new.line><li>Forms<li.new.line><li>Animations<li.new.line><li>Contributions to an open-source project<li.new.line><ul.new.line><p>This article focuses on learning how to build a functional portfolio using HTML, CSS and Bootstrap that looks great regardless of the device used to display it.<p.new.line><h2 id="3kb0q">The Tools You‚Äôll Use<h2.new.line><p>To be able to work on a portfolio with these tools, you at least need to be familiar with how they each work. This section will shed a bit of light on getting started if you haven&#x27;t used them before or refresh your memory from past projects.<p.new.line><h3 id="4al2">HTML<h3.new.line><p><u><a href="https://www.w3schools.com/html/html_intro.asp" target="_blank" rel="noopener noreferrer">HTML</a></u>, or Hyper Text Markup Language, is the standard markup language for Web pages. HTML consists of elements that tell the browser how to display the content. In a nutshell, we use HTML to define the building blocks of web pages.<p.new.line><h3 id="eti6p">CSS<h3.new.line><p><u><a href="https://www.w3schools.com/css/" target="_blank" rel="noopener noreferrer">CSS</a></u>, or Cascading Style Sheets, is a programming language used to style web pages and HTML. It describes how to display HTML elements. We‚Äôll leverage the<a href="https://getbootstrap.com/docs/5.2/getting-started/introduction/" target="_blank" rel="noopener noreferrer"> <u>Bootstrap</u></a> CSS style to make development faster.<p.new.line><h3 id="1l73u">Bootstrap<h3.new.line><p>Bootstrap is a toolset for the front end and includes features that you can use to style nearly anything.<p.new.line><h2 id="ed4v0">Building a Portfolio<h2.new.line><h3 id="3dut8">Folder and Files<h3.new.line><p>You can get the starter files for this project on<a href="https://github.com/Nessii-green/Goodnessportfolio" target="_blank" rel="noopener noreferrer"> <u>Github</u></a>.
