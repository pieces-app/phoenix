# Using CSS to Create a Skeleton Screen
> Wed Apr 05 2023 16:53:15 GMT+0000 (Coordinated Universal Time)
<p>The web has recently undergone improvements that benefit both consumers and developers. Because new methods of enhancing user experiences continue to emerge over time, user experience is a major focus that goes beyond present metrics.<p.new.line><p>Skeleton displays are one component of UX improvement. In this article, we’ll be covering the purpose of skeleton loaders, their significance, and how to create one for a website using React and basic CSS.<p.new.line><h2 id="5jmcp">What is a Skeleton Screen?<h2.new.line><p>An animated placeholder known as a &quot;skeleton screen&quot; simulates a website&#x27;s design while data is being loaded. Skeleton screens inform the user that some content is loading while also, and perhaps more crucially, indicating whether an image, text, card or other type of data is loading.<p.new.line><p>Because the user is aware of the type of content loading before it manifests, the page appears to load more quickly. Perceived performance is what we&#x27;re talking about here.<p.new.line><p>Here are some illustrations of skeleton displays from Medium and Facebook:<p.new.line><figure class="w-richtext-figure-type-image w-richtext-align-center" data-rt-type="image" data-rt-align="center"><div><img alt="A Medium skeleton screen." src="https://uploads-ssl.webflow.com/6143afec68f555387049efb3/63a1d17e8f7d334ce21a61ab_image4_a11ea04ce848a98507f5371275c78fa5_800.png"><div.new.line><figcaption>Medium home feed loading state</figcaption><figure.new.line><figure class="w-richtext-figure-type-image w-richtext-align-center" data-rt-type="image" data-rt-align="center"><div><img alt="A Facebook skeleton screen." src="https://uploads-ssl.webflow.com/6143afec68f555387049efb3/63a1d17e8f7d33c9411a61ac_image5_55f3297010408508bc0074de9526a084_800.png"><div.new.line><figcaption>Facebook home feed loading state</figcaption><figure.new.line><p>The gray shapes indicate how the data will be shown once it has loaded. These images are replaced once the data from the server actually arrives.<p.new.line><p>Skeleton loaders seldom improve efficiency or loading speed. They exist solely to provide the visitor with something to look at and the impression that the website is loading more quickly.<p.new.line><h3 id="5n160">Notable Qualities of Skeleton Loaders<h3.new.line><ul><li>They seem to move more quickly and are easier to utilize. A better user experience and higher conversion rates are both provided by improved perceived performance.<li.new.line><li>They give the impression of quickness and quick loading.<li.new.line><li>Users have no indication of what is loading or how long it will take with spinners and loaders, which can be a problem.<li.new.line><li>Since the load time is unknown, using spinners or loaders puts the user in a state of uncertainty.<li.new.line><li>Users might be tempted to move on rather than wait through skeleton screens.<li.new.line><ul.new.line><h2 id="djgr7">Build a Skeleton Loader with React<h2.new.line><p>For the purpose of this article&#x27;s tutorial, we&#x27;ll build a page that lists blog articles as cards.
